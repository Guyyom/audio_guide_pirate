What is Audio Guide Pirate?

Audio Guide Pirate is a proof of concept, it allows users to obtain an .mp3 file of the main description of any Wikipedia pages, as long as this latter exists.

How to use it?

This tool uses the API from Wikipedia, in other words if you are travelling in a city or if you want to obtain more information about a painting, just follow the below steps:

1. Select the Wikipedia page of the city you are visiting, or anything else you would like to obtain (Go on Wikipedia : here);

2. Copy the "title" of the article you chose, for instance from Mona Lisa Wikipedia's page, the title is : "Mona Lisa";

3. Paste the title into the form on Home page and select the required language (must be similar from the Wikipedia page);

4. Press "Send Request"

5. If all the values are correct you shall be redirected to the .MP3 files automatically (Please note the process might take few seconds to execute);
